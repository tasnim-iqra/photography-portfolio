# Photography Portfolio Website

This is a simple, responsive, one-page portfolio website for a photographer. It is built with HTML, Tailwind CSS, and vanilla JavaScript.

## Features

*   Clean, modern design
*   Responsive layout
*   Photo gallery
*   About Me section
*   Contact section with email obfuscation

## How to View

1.  Clone this repository to your local machine.
2.  Open the `index.html` file in your web browser.

## GitHub Pages

This website is designed to be hosted on GitHub Pages. To deploy this website to GitHub Pages, you will need to:

1.  Create a new repository on GitHub.
2.  Push the files in this directory to the new repository.
3.  In the repository's settings, go to the "Pages" section.
4.  Select the `main` branch as the source and click "Save".
5.  Your website will be available at `https://<your-username>.github.io/<your-repository-name>/`.
